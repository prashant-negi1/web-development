What is an HTML Tables?

    • A table is used to display tabular data in rows and columns.
    • Semantic: represents data, not layout.
    • Container tag: <table>.

    Basic Table Structure

    1. <table> – Table container
    2. <tr> – Table row
    3. <th> – Table header cell
    4. <td> – Table data cell
    5. <caption> – Table title
    6. <thead>, <tbody>, <tfoot> – Group header, body, footer rows
    7. colspan – Merge columns
    8. rowspan – Merge rows
    9. Use CSS for borders, padding, and styling.
    10. Tables are for **data**, not layout.
